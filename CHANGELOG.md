# 更新日志

## [0.0.1] - 2025-11-13

### 新增功能

#### 1. 目录文件自动补全
- 输入 `/` 触发补全，显示当前目录下的所有文件和子目录
- 支持多级嵌套目录的连续导航
- 选择目录后自动添加 `/` 并继续触发下一级补全
- 自动查找项目根目录（包含 debugtalk.py 的目录）
- 目录优先显示，文件按字母排序

#### 2. debugtalk.py 函数自动补全
- 输入 `$` 时自动列出所有公开函数
- 自动插入完整的 `${function_name()}` 表达式
- 支持参数补全：
  - 无参数函数直接插入
  - 有参数函数使用 Snippet 模式
  - 参数高亮显示，Tab 键跳转
- 自动过滤私有函数（以 `_` 开头）
- 支持带默认值和类型注解的参数解析
- 实时解析，无需重启

### 技术实现

- TypeScript 5.3
- VS Code Extension API 1.85
- 正则表达式解析 Python 函数定义
- 文件系统遍历实现路径补全
- SnippetString 实现参数跳转

### 已知限制

- 仅支持 YAML 文件
- 仅解析 debugtalk.py 中的简单函数定义
- 项目根目录向上查找最多 10 层
